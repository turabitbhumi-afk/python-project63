# Web Scraping Projects

Collection of web scraping projects built with Python.

## Projects

### 1. Books Scraper
- Site: books.toscrape.com
- Scrapes: title, price, stock, rating
- Tech: Firecrawl, Python
- Output: CSV

### 2. IMDb Top 250
- Site: imdb.com/chart/top
- Scrapes: rank, title, year, rating
- Tech: Firecrawl, regex, Python
- Output: CSV

### 1. E-commerce Playwright Scraper (test.py)
Scrapes laptop names and prices from a JS-rendered e-commerce site.
Handles pagination automatically.
- Tech: Python, Playwright
- Output: laptops.csv
